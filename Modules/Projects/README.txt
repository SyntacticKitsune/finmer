The following modules are provided:

Core
	Contains all official game content.

Example01 - Injection Demo
	Provides a very basic demo for the scene patching system, which enables a module to inject new content into scene assets that were originally provided by another module.
	This module adds a 'hello world' button to the notice board in North Finmer's town square.

Example02 - Custom Combat Arena
	Provides an overview of the different functions of the V2 combat system, and provides a testing bed for quickly running combats with different combinations of participants or equipment.
	This module adds a 'Custom Combat' button to the game's opening scene, from where a combat scenario can be configured and launched.
